# 🤝 How to Contribute

We want this to be the loudest repository on GitHub. If you have a strategy that worked, add it!

### Ways to Contribute:
1. **Submit a Success Story:** Add your win to the `HALL_OF_FAME.md`.
2. **Share a Tool:** Found a new free AI app? Add it to `FREE_AI_RESOURCES.md`.
3. **Refine a Script:** If you found a better way to phrase an advocacy move, let us know.

### How to do it:
* Fork this repository.
* Make your changes.
* Submit a **Pull Request** with a brief description of what you added.

**Stay Unhinged. Stay Loud.**
